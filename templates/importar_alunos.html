O arquivo deve conter as seguintes colunas:
  • nome
  • email
  • cpf
  • rg
  • data_nascimento (formato: DD/MM/YYYY)
  • responsavel
  • contato
  • contato_emergencia

Instruções
  1. Prepare um arquivo Excel (.xlsx) com as colunas listadas acima
  2. Certifique-se de que os dados estão formatados corretamente:
    • CPF: apenas números ou no formato XXX.XXX.XXX-XX
    • Data de nascimento: no formato DD/MM/YYYY
    • Telefones: apenas números ou no formato (XX) XXXXX-XXXX
  3. Faça upload do arquivo usando o formulário acima
  4. O sistema irá:
    • Verificar duplicidade de CPF, email e RG
    • Validar o formato dos dados
    • Gerar senhas aleatórias para cada aluno
    • Enviar emails com as credenciais de acesso
